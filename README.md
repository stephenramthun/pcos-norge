Website for PCOS-Norge built with [Next](https://nextjs.org) and [Sanity](https://www.sanity.io)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
